
# Personal Portfolio

A minimal, responsive developer portfolio built with Next.js 16, React 19, and Tailwind CSS 4 to showcase my work and projects. Includes dark/light mode, MDX-powered blogs, smooth animations, and SEO optimization.


![Portfolio Preview](/public/Hero.png)



## Features

- **Next.js 16** with App Router
- **React 19** for modern React features
- **Tailwind CSS 4** for styling
- **shadcn UI** components
- **Dark/Light** mode
- **Responsive** design
- **MDX** for blog posts and content
- **Contact Form** with email integration
- **SEO** optimized
- **TypeScript** for type safety
- **GitHub Calendar** integration
